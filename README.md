# 🎉 Personal Space

A beautiful and interactive personal memory website built to celebrate someone special. This project lets you create a personalized experience by combining photos, videos, heartfelt messages, memorable moments, and background music into a single responsive webpage.

## ✨ Features

* 🎬 Hero section with personalized image/video
* 💖 Custom welcome message
* 📖 Story & memorable moments section
* 🖼️ Responsive photo gallery
* 🎵 Background music support
* 📱 Fully responsive design
* ⚡ Smooth scrolling and animations
* 🎨 Easy to customize

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/thevishalmisra/Personal-gallery.git
```

Go to the project folder:

```bash
cd Personal-gallery
```

Open `index.html` in your browser or run it using VS Code Live Server.

## 🎨 Customization

You can easily personalize the website by:

* Replacing images and videos
* Editing names and messages
* Adding your own memories
* Changing colors, fonts, and animations
* Updating background music
